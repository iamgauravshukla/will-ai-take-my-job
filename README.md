# AI Take Job

AI job automation risk analyzer with resume parsing, role-level risk pages, and shareable analysis results.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Guides

- Setup guide: [docs/setup-guide.md](docs/setup-guide.md)
- Project guide: [docs/project-guide.md](docs/project-guide.md)

```bash
npm run seed:jobs
npm run generate:seo
npm run build
npm run start
```
